# number-guessing-game

This project is a number guessing game where the program generates a random number and the user attempts to guess it, receiving feedback like “too high” or “too low” until they win.

||||||||||||||||🎮 How It Works ???????????????????????????
1------ The program generates a random number within a specific range.
2-------The player is asked to guess the number.
3-------------After each guess:
*******The program tells if the guess is too high or too low.
4----------The game continues until the player guesses correctly.
5-------------The total number of attempts is displayed at the end.
